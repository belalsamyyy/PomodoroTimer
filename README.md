# PomodoroTimer

Using Timer class and enums to create simple pomodoro timer with with 4 sessions :-
- [X] focus 25 mins -> short break 5 mins 
- [X] focus 25 mins -> short break 5 mins 
- [X] focus 25 mins -> short break 5 mins 
- [X] focus 25 mins -> long break 15 mins
